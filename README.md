# 🐄 Pineapple QurbaniHat – Livestock Booking Platform

## 📌 Project Overview

Pineapple QurbaniHat is a modern livestock marketplace web application where users can explore animals for Qurbani such as cows and goats. Users can view detailed information about each animal and place a booking after authentication.

---

## 🎯 Purpose

The purpose of this project is to provide a simple and user-friendly platform for browsing and booking Qurbani animals online. It demonstrates authentication, private routes, and interactive UI features in a real-world scenario.

---

## 🌐 Live Website

🔗 https://livestock-booking-5dpp.vercel.app/

---

## ✨ Key Features

* 🔐 User Authentication (Email & Google Login)
* 📝 User Registration System
* 🐐 Browse All Animals with Details
* 🔎 Sort Animals by Price
* 📄 Animal Details Page (Private Route)
* 📦 Booking Form with Success Toast Notification
* 👤 My Profile Page (View User Info)
* ✏️ Update Profile Information (Name & Image)
* 🎬 Animation using Animate.css
* 📱 Fully Responsive (Mobile, Tablet, Desktop)
* 🚫 Custom Not Found Page
* ⏳ Loading Indicator for Data Fetching
